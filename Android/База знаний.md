## Как подключить kapt для DataBinding
#### build.gradle.kts (для модуля)
plugins {  
    kotlin("kapt")  
}
android {  
    buildFeatures {  
        dataBinding = true  
        // for view binding:  
        // viewBinding = true
    }
}
dependencies {  
    kapt("com.android.tools.build:gradle:8.5.1")  
}
#### settings.gradle.kts
pluginManagement {  
    plugins {  
        kotlin("kapt") version "2.0.20"  
    }
}
